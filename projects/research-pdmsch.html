Xenopus Embryo Sorter Design

Skills Gained

Mechatronics Microfluidics SLA Printing FDM Printing PDMS Mold Design Hardware Integration

Description

Situation: Xenopus eggs often come in batches of hundreds or thousands, requiring each egg to be checked and filtered out if deemed useless. This process was previously done manually, involving hours of tedious labor.

Task: As the sole mechanical engineer, I was tasked with designing the hardware components for a mechatronics project aimed at automating the sorting process of these eggs.

Summary

Action: I designed the hardware for the system for the ~1mm wide eggs that flow through the chip. Using a combination of sensors and pressure regulators, the eggs are stopped, photographed on both sides, and sorted with a machine learning model. As they flow out, sensors track each egg's path, and a quake valve sorter mechanism, selected after iterating through various sorting mechanisms, separates them.

Result: The system includes a transparent chip made of PDMS, cast from an SLA printed mold, housed between two mirrors. This setup allows a camera to capture images of both the top and bottom of the chip, enabling the machine learning model to categorize the eggs effectively.

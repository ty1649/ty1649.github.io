FSAE Suspension Design

Skills Gained

CAD Modeling FEA Analysis Project Management Technical Documentation Manufacturing Processes

Problem Statement

The Formula SAE team needed to design and implement a new suspension system that would improve vehicle handling while maintaining compliance with competition regulations. The system needed to be lightweight, adjustable, and manufacturable within our constraints.

My Work

Led the suspension design team through the complete development cycle:

  • Conducted kinematic analysis using simulation software
  • Created detailed CAD models of all components
  • Performed FEA to validate design decisions
  • Coordinated with manufacturing team for production
  • Developed testing protocols for validation

Results

The new suspension system achieved several key improvements:

  • 20% reduction in overall weight
  • Improved cornering performance by 15%
  • Enhanced adjustability for different track conditions
  • Successfully passed all technical inspections

Lessons Learned

This project provided valuable insights into:

  • Balancing theoretical design with practical constraints
  • Importance of iterative testing and refinement
  • Value of clear documentation and communication
  • Managing timeline and resource constraints

Fingerprint Door Unlocker

Skills Gained

Arduino 3D Printing CAD Design Mechatronics Embedded Systems

Problem Statement

Living in a dorm means carrying a key card everywhere, which can be inconvenient and time-consuming when trying to enter your room. The traditional key card system requires fumbling through pockets or bags, potentially dropping items, and swiping multiple times if the reader doesn't work on the first try. I wanted to create a more efficient and user-friendly way to unlock my dorm door that would eliminate the need for physical credentials while maintaining security.

My Work

I designed and developed a complete door unlocking system that uses fingerprint authentication for secure and convenient access. The project combined hardware design, 3D printing, and Arduino programming to create a practical solution.

  • Designed a compact housing that could be mounted on the door without permanent modifications
  • Implemented a lead screw mechanism for smooth, reliable linear motion to actuate the door handle
  • Created and 3D printed all structural components to ensure perfect fit with the existing door hardware
  • Programmed an Arduino to interface with the fingerprint sensor and control the motion system
  • Developed a power-efficient system that could operate on battery power for extended periods

Results

The fingerprint door unlocker proved to be a significant improvement over the traditional key card system, providing both convenience and efficiency gains.

  • Reduced time required to enter dorm by 50% compared to using a key card
  • Successfully eliminated the need to carry and manage physical credentials
  • Created a secure system that only responds to registered fingerprints
  • Designed a non-permanent installation that doesn't damage door hardware
  • Implemented a fail-safe mechanism that allows for manual operation if needed

Lessons Learned

This project provided valuable experience in combining mechanical design with electronics and programming to create a practical solution to an everyday problem.

  • Learned the importance of designing for reliability in daily-use mechanical systems
  • Gained experience in optimizing 3D printed parts for strength and durability
  • Developed skills in integrating biometric sensors with mechanical actuators
  • Understood the challenges of creating user-friendly security systems
  • Improved ability to iterate on designs based on real-world testing and feedback
